<h2>Hur kan jag spela?</h2>

Steg 1. Navigera till den gröna knappen <> code och klicka.
Steg 2. Klicka på knappen höger om URL'en (den lilla ikon med två papper).
Steg 3. Öppna ditt valda kodprogram.
Steg 4. Navigera till din terminal eller ctrl + ö (för vsCode med svensk layout).
Steg 5. Skriv in "git clone URL_DU_KOPIERADE" sen Enter (för bash).
Steg 6. Öppna projekt-mappen (. code).
Steg 7. Have fun.

<h2>Hur gör jag?<h2>

Steg 1. När du känner dig redo, klicka på start.
Steg 2. Se "mullvadarna" dyka upp i rutorna.
Steg 3. KLICKA PÅ DEN! KLICKA MED MUSEN! KLICKA INNAN DEN FÖRSVINNER!!
Steg 4. Repetera steg 1 och 2 tills tiden är ute.
Steg 5. repitera steg 1 - 4 tills du känner dig nöjd.

Klickad "mullvad" = Poäng.
Missad "mullvad" = Inga poäng.
